# Office Tileset Assets

This folder requires the **LimeZu Modern Office** tileset (paid asset).

## How to obtain

1. Purchase "Modern Office" by LimeZu from [itch.io](https://limezu.itch.io/modernoffice)
2. Download and extract the archive
3. Copy these files into this folder:

```
furniture.png    <- from 2_Modern_Office_Black_Shadow/Modern_Office_Black_Shadow.png  (256x848)
room-builder.png <- from 1_Room_Builder_Office/Room_Builder_Office_16x16.png          (256x224)
```

The game will show an empty/broken state if these PNGs are not present.

## Sheet specs

- **furniture.png**: 256x848, 16x53 grid of 16px tiles, no margin
- **room-builder.png**: 256x224, 16x14 grid of 16px tiles, no margin
